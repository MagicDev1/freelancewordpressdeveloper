A) Introduction - top

Invoicer is Web Application to generate PDF invoices, Orders, Quotes and Receipts right in your browser within seconds with out writing a single line of code.

Just need to put your information, press the button and you will get a beautifully designed PDF invoice downloaded.

You can also send the invoice directly to your client without downloading it. Simply click on the Send Email button and put your and your clients email address with a custom message.

And your client will receive a beautifully designed HTML email with the PDF file attached with it.

It's very easy to use and install.

B) Installation - top

Invoicer is a software that runs directly into your web browser and no need of any installation. You just need a web server and you are good to go.

The downloaded package has following folders in it:

  • _Documentation - It has all the documentation and instructions for the usage of the App (You are here)
  • WWW - This is the folder which has the App files in it
  • Changelog - Detail of updates.

Now head to WWW folder and get invoicer.zip folder of the application from it.

Unzip invoicer.zip folder and than upload the invoicer folder (uploading folder must have files like index.php, send.php and generator.php in it) and all content to www or public html directory of your webserver.

After upload the contents on web server. You can Surf to http://[yourwebsite]/invoicer to access the application.

That's it! You are done with the installation. Now you can generate and download unlimited PDF invoices right from your web browser.

D) Files & Folders - top

  • css - This folder has all the stylesheets of the application. This app is built with bootstrap 3 so you can easily change the look of the app.
  • fonts - This folder has icon fonts used in the front end forms.
  • images - This folder has some images used in the app.
  • includes * - This folder has all the required Php files and libraries of the application. Important: Do not change anything in the files of this folder.
  • js * - This folder has all the javascript files used in the app. Important: Do not change anything in the files of this folder.
  • uploads - Users uploaded custom logos will be uploaded in this folder.
  • index.php - In this file you can found all the code of the form that input the values for PDF details.
  • send.php - This file has all the code of the Invoice Sending front end.
  • generator.php - This file is responsible for generating PDF file based on all the details added in the form.
